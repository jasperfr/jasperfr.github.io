About

Bit Breaker is an incremental idle game with tons of automation.
This build however is currently in an early alpha! You can help by testing it out and giving me some feedback.
This game is intended to be free. Free as in should there ever be a mobile game made, there won't be any room for pay to progress stuff.
Although if you'd like, you can still donate! (obviously not right now since the game is at an early alpha stage)
